# 💖 Nosso Amor

Um projeto web interativo feito para acompanhar e celebrar um relacionamento de forma divertida e personalizada.

## ✨ Funcionalidades

* 🔐 Tela de acesso com código secreto
* ⏱️ Contador em tempo real do relacionamento
* 💬 Chat com mensagens prontas
* 🤖 Respostas automáticas
* 🎮 Jogo interativo (quiz do casal)
* 💾 Salvamento automático no navegador (LocalStorage)
* 🔄 Botão para resetar tudo

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript (puro)

## 📱 Objetivo

Criar uma experiência romântica e interativa, funcionando como um “mini app” de casal diretamente no navegador.

## 🚀 Como usar

1. Abra o arquivo `index.html` no navegador
2. Digite o código de acesso
3. Interaja com o chat ou jogue o quiz

## 📌 Observações

* Os dados são salvos apenas no navegador (não sincroniza entre dispositivos)
* Projeto focado em uso pessoal
